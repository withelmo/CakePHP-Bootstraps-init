# Bootstraps-init for CakePHP

## How to use

1. Download the files

  ```bash
  cd Path/app/Config
  git clone git@github.com:withelmo/CakePHP-Bootstraps-init.git Bootstraps
  ```

2. Initilize

  ```bash
  cd Bootsraps
  sh init.sh
  ```
  
3. Add environment config
  (example) .htaccess

  ```bash
  CAKE_ENV_MODE='development'
  ```
